# Setup virtual environment and dependencies
```bash
python -m venv .venv
source .venv.bin/activate
pip install -r requirements.txt
```
# Install dependencies

```bash
pip install -r requirements.txt
```

# Run tests

```bash
python tests.py
```

# Files

- tests.py: Tests.
- scrabble.py: Game logic.
- dictionary.py: valid words
